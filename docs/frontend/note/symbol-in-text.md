# 文本中的符号

> 换行符、回车符、\n、\r、\n\r 

## html

我们都知道  `<br>` 表示行级标签的换行。

## text

多数的符号都是用在文本中，下面一一了解。

```bash
# tab 制表符
\t

# return 回车
\r

# newline 换行
\n

# 换行+回车
\n\r
```

### windows

'\r' 回车，回到当前行的行首，而不会换到下一行，如果接着输出的话，本行以前的内容会被逐一覆盖；

'\n' 换行，换到当前位置的下一行，而不会回到行首；  

'\r\n'下一行，跳转到下一行的行首。

### mac

Mac系统里，每行结尾是“<回车>”，即"\r"

### linux

Unix系统里，每行结尾只有“<换行>”，即"\n"
