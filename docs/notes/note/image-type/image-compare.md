## 常见图片格式对比

> 关键词：jpg、jpeg、png、gif 。。。

## 名词

| 对比   | 解释      | 对比   | 解释            |
| ------ | --------- | ------ | --------------- |
| 有损   |           | 无损   |                 |
| 索引色 | 256种颜色 | 直接色 | 2的32次方种颜色 |
| 位图   | 像素      | 矢量图 | 算法            |

## 格式

### BMP

- 无损
- 索引色/直接色
- 位图

### JPEG

- 有损
- 直接色
- 位图

### JPG

- jpg 和 jpeg 图片没有任何区别。

### png-8（索引色版本）

- 无损
- 索引色
- 位图

### png-24（直接色版本）

- 无损
- 直接色
- 位图

### GIF

- 有损
- 索引色
- 位图

### SVG

- 无损
- 矢量图

### WebP

- 有损/无损
- 直接色
- 位图
